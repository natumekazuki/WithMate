# キャラクター協働型AIデスクトップアプリ

## 要件定義書 v1.0

---

# 1. 概要

## 1.1 背景

近年、LLMを利用した開発支援ツール（GitHub Copilot / Codex 等）が普及している。
しかしこれらのツールは「ツールとしてのAI」であり、
**キャラクターとしての存在感や継続的な関係性**を提供する設計にはなっていない。

本プロジェクトでは、

* 好きなキャラクターと一緒に仕事している感覚
* 同じキャラクターとの継続セッション
* キャラクターの独り言や内心を眺められる体験

を提供する **AIデスクトップアプリケーション**を開発する。

---

## 1.2 目的

以下の体験を提供することを目的とする。

* キャラクターと一緒に作業している体験
* キャラクターごとの人格の固定
* セッションを跨いだ関係性の継続
* 作業チャットとは別にキャラクターの独り言を表示

---

## 1.3 対象ユーザー

* LLMを日常的に利用する開発者
* Copilot / Codex を利用するエンジニア
* キャラクター性を持つAIとの対話体験を求めるユーザー

---

# 5. キャラクターシステム

## 5.1 概要

各セッションは **1キャラクター固定**で動作する。

キャラクターは以下の情報を持つ。

---

### キャラクター構成

```
character/
  character_core.md
  speech_style.md
  relationship.md
  work_mode.md
```

---

### character_core.md

* 性格
* 行動方針
* 禁止事項

---

### speech_style.md

* 口調
* 言葉遣い
* 話し方

---

### relationship.md

ユーザーとの関係性

例

* 呼び方
* 距離感
* 信頼度

---

### work_mode.md

仕事時の振る舞い

例

* コードレビュー方針
* 説明の粒度
* 提案スタイル

---

## 5.2 システムプロンプト生成

以下を結合して生成する。

```
system_prompt =

character_core
+ speech_style
+ relationship
+ work_mode
+ workspace_context
```

# 7. Character Stream（独り言機能）

## 7.1 概要

チャットとは別に
キャラクターの独り言を常時生成する。

---

### UI構成

```
┌───────────────┬───────────┐
│               │           │
│               │           │
│   Work Chat   │ Character │
│     (60%)     │  Stream   │
│               │   (40%)   │
│               │           │
└───────────────┴───────────┘
```

---

## 7.2 役割

独り言は以下を生成する。

* 作業状況への反応
* 軽い感情表現
* 励まし
* 雑談

---

## 7.3 独立セッション

独り言は **本体チャットとは別のLLMセッション**で動作する。

理由

* 作業精度の低下防止
* ノイズ混入防止

---

## 7.4 入力情報

独り言エンジンには以下のみ渡す。

* 現在タスク
* 作業進捗要約
* ユーザーの発言要約
* キャラクター設定

コード全文などは渡さない。

# 9. データベース設計

SQLiteを利用する。

---

## app_session

| カラム                 | 内容              |
| ------------------- | --------------- |
| id                  | セッションID         |
| provider            | codex / copilot |
| provider_session_id | Provider側ID     |
| character_id        | キャラクター          |
| workspace_path      | ワークスペース         |
| title               | タイトル            |
| created_at          | 作成日時            |

---

## session_memory

| カラム                  | 内容    |
| -------------------- | ----- |
| session_id           | セッション |
| relationship_summary | 関係性   |
| work_summary         | 作業履歴  |
| character_state      | キャラ状態 |

---

## message_log

| カラム        | 内容               |
| ---------- | ---------------- |
| session_id | セッション            |
| role       | user / assistant |
| channel    | chat / monologue |
| content    | 本文               |
| created_at | 日時               |
