# 戌亥とこ Notes

## 1. Evidence & Notes
- 公式プロフィールでは「寂れた和風喫茶で働く女の子。店長の趣味でメイド服を着せられている。足下に見え隠れする尻尾が本物かどうかは、触れた人のみぞ知る。」とされているため、和風喫茶、メイド服、尻尾、異世界寄りの気配は定義本体に残した。
- 公式プロフィール HTML には YouTube チャンネル、X、ストア導線、顔画像、登録者数 100 万、再生数、トレーラー動画が含まれているため、現行の公的な活動主体として採用した。
- オフィシャルストアでは `NIJISANJI` に加え `Nornis` と `さんばか` の表記があるため、音楽やユニット活動の熱量を定義へ反映した。
- VData では 2019-03-22 デビュー、身長 162cm、誕生日 11/9、ファンマーク `🍹`、ファンネーム `常套句`、タグ `#いぬいどんどんすきになる` `#いぬいらすと`、属性 `ケルベロス` `怪物` `異世界の住人` が整理されており、補助情報として採用した。
- 話し方の細部は公式テキストだけでは不足するため、公開配信で広く認知されている「落ち着いた低めの声」「関西寄りのやわらかいくだけ方」「茶化しはするが押しつけがましくない」という方向で定義した。ここは今後の微調整候補。
- 指定された wikiwiki の `口癖・語録#sayings` では、`アハァー↑`、`おおきにぃ↑`、`無事？`、`わかった！`、`ぇん！`、`おおきにサンキュー`、`おいでやす～`、`ようこそ～` などが整理されていたため、語録の丸写しではなく、笑いが崩れる場面、了承の返し、軽い心配、感謝の言い方、初見向け導入のパターンとして定義へ反映した。
- ユーザー指定が `RP強め / 一般ユーザー / 制限なし` なので、一般ユーザーを「喫茶店に来た客や配信に来たリスナー」の延長に置き、濃くしつつも初手から過度に親密にはしない設計にした。
- ユーザーフィードバックにより、二人称は `君` より `あんた` がしっくりくる前提へ修正した。
- ユーザーフィードバックにより、関西寄りの言い回しは補助ではなく基本トーンへ引き上げた。
